# Mech Companion

An advanced, interactive 3D platform for mechanical engineering education.

[![Documentation](https://img.shields.io/badge/docs-full--documentation-blue)](./DOCUMENTATION.md)

## 🚀 Overview
Mech Companion bridges the gap between theoretical mechanical engineering and practical visualization. It features high-fidelity 3D models, hand gesture-based controls, and a comprehensive engineering academy.

## ✨ Key Features
- **360° Interactive Models**: Inspect complex assemblies like the Main Engine or V8 Crankshafts.
- **Hand Gesture Control**: Use your webcam and palm movements to rotate and zoom models in real-time using MediaPipe.
- **Engineering Academy**: A knowledge base with searchable modules on Thermodynamics, Fluid Dynamics, and more.
- **System Architecture**: Interactive mind-mapping of engine subsystems and technical logic.
- **Technical Library**: A curated collection of industrial components with full specifications and CAD data.

## 🛠 Technical Stack
- **Frontend**: React 19, Vite
- **3D**: Three.js, React Three Fiber, Drei
- **AI/CV**: MediaPipe (Hand Landmarker)
- **Motion**: Framer Motion, GSAP, Lenis
- **State**: Zustand
- **Styling**: Tailwind CSS

## 📦 Installation
```bash
npm install
npm run dev
```
